get /ingredients/search
Search for ingredient (searchPayload)
Search for simple whole foods (e.g. fruits, vegetables, nuts, grains, meat, fish, dairy etc.).

Query parameters

query (optional)
Query Parameter — The partial or full ingredient name. <br/> <br/><em>Example: apple</em>.
addChildren (optional)
Query Parameter — Whether to add children of found foods. <br/> <br/><em>Example: <code>true</code></em>.
minProteinPercent (optional)
Query Parameter — The minimum percentage of protein the food must have (between 0 and 100). <br/> <br/><em>Example: 35</em>.
maxProteinPercent (optional)
Query Parameter — The maximum percentage of protein the food can have (between 0 and 100). <br/> <br/><em>Example: 90</em>.
minFatPercent (optional)
Query Parameter — The minimum percentage of fat the food must have (between 0 and 100). <br/> <br/><em>Example: 10</em>.
maxFatPercent (optional)
Query Parameter — The maximum percentage of fat the food can have (between 0 and 100). <br/> <br/><em>Example: 90</em>.
minCarbsPercent (optional)
Query Parameter — The minimum percentage of carbs the food must have (between 0 and 100). <br/> <br/><em>Example: 10</em>.
maxCarbsPercent (optional)
Query Parameter — The maximum percentage of carbs the food must have (between 0 and 100). <br/> <br/><em>Example: 90</em>.
metaInformation (optional)
Query Parameter — Whether to return more meta information about the ingredients.<br/> <br/> <em>Example: <code>false</code></em>.
intolerances (optional)
Query Parameter — A comma-separated list of intolerances. All recipes returned must not contain ingredients that are not suitable for people with the intolerances entered. A comma-separated list of intolerances. All recipes returned must not contain ingredients that are not suitable for people with the intolerances entered. See a full <a href="https://spoonacular.com/food-api/docs#Intolerances">list of supported intolerances</a>. <br/> <br/><em>Example: Egg</em>.
sort (optional)
Query Parameter — The number of results to skip (between 0 and 990).<br/> <br/><em>Example: Calories</em>.
sortDirection (optional)
Query Parameter — The direction in which to sort. Must be either 'asc' (ascending) or 'desc' (descending).<br/> <br/><em>Example: asc</em>.
language (optional)
Query Parameter — Language code, either &quot;en&quot; or &quot;de&quot;. default: en
offset (optional)
Query Parameter — The number of results to skip (between 0 and 990)..<br/> <br/><em>Example: 0</em>.
